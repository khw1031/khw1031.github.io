# VitePress 블로그 활용하기

VitePress를 사용하면 정적 사이트 생성(SSG)의 장점을 활용할 수 있습니다.

## 장점

1. **빠른 속도**: Vite 기반으로 매우 빠른 개발 경험
2. **타입 안정성**: TypeScript 완벽 지원
3. **Vue 활용**: Vue 컴포넌트를 마크다운에 직접 사용 가능
4. **테마 커스터마이징**: 유연한 테마 시스템

## 포스트 작성 방법

포스트는 `home/posts/` 디렉토리에 마크다운 파일로 작성합니다:

```markdown
---
title: "포스트 제목"
date: YYYY-MM-DD
---

# 본문 작성
```

간단하죠? 🎉
