# VitePress 블로그에 오신 것을 환영합니다

이것은 VitePress로 만든 블로그의 첫 번째 포스트입니다.

## 주요 기능

- ✅ 마크다운 기반 작성
- ✅ 자동 포스트 목록 생성
- ✅ 날짜순 정렬
- ✅ 빠른 빌드 속도

## 코드 예제

```typescript
function hello(name: string) {
  console.log(`안녕하세요, ${name}님!`);
}
```

VitePress는 강력한 마크다운 기능과 Vue 컴포넌트를 지원합니다.
