# Interview Questions

## MCP

- MCP는 기존 API 호출이나 plugin 방식과 무엇이 다른가?
- MCP tool schema를 설계할 때 LLM이 잘 쓰게 하려면 무엇을 고려해야 하는가?
- Cursor, Claude Code, GitHub Copilot에서 MCP 연동 차이는 무엇인가?

## OpenAPI

- OpenAPI 3.0과 3.1의 차이는 무엇인가?
- `$ref` component를 별도 tool로 분리한 이유는 무엇인가?
- requestBody와 responses에서 타입을 생성할 때 주의할 점은 무엇인가?

## Context Strategy

- 전체 Swagger 문서를 한 번에 넣지 않고 drill-down으로 조회한 이유는 무엇인가?
- 토큰 절약과 정확도 사이의 tradeoff는 무엇인가?

## Security
