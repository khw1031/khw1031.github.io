# Rewrite Notes

## Target Positioning

## Proposed Structure

1. TL;DR
2. 문제: 표준 제안자 중심 평가의 한계
3. 오픈소스 모델의 시사점
4. 사내 표준 확산 역할 모델
5. 기여 포인트 예시
6. 기대 효과와 리스크
7. 결론
