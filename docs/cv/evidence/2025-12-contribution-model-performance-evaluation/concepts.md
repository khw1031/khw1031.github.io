# Concepts

## Contribution Model

- 표준의 성공을 만든 사람뿐 아니라 검증하고 채택하고 개선한 사람에게도 기여를 부여하는 모델.

## Open Source Analogy

- Maintainer는 방향과 품질을 책임진다.
- Contributor는 issue, review, documentation, bug report, adoption case로 생태계 품질을 높인다.

## Incentive Design

- 보상 체계는 원하는 행동을 만든다.
- 잘못 설계하면 점수 최적화가 실제 가치 창출보다 앞설 수 있다.

## CV Relevance

- 기술 구현 프로젝트보다 조직 생산성/표준 확산 관점의 리더십 근거로 활용 가능.
