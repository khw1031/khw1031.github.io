# Metrics

## Known Metrics

현재 포스트는 정량 지표보다 조직 운영 모델 제안에 가깝다.

## Verification Needed

- 실제 조직에 제안되었는지.
- Beta Tester, Early Adopter, Contributor 역할이 실제 운영되었는지.
- 기여 포인트가 KPI나 평가에 연결된 사례가 있는지.

## CV-Safe Summary

표준 확산의 성과를 Maintainer 중심이 아니라 Beta Tester, Early Adopter, Contributor까지 포함해 인정하는 Contribution 모델 제안.
