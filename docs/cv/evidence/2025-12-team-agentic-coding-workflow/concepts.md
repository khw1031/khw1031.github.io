# Concepts

## Agentic Coding

- 목표를 받은 에이전트가 계획, 수정, 테스트, 오류 수정까지 수행하는 작업 방식.
- 팀 환경에서는 개인의 프롬프트 차이가 코드 품질 편차로 이어질 수 있다.

## Grounding Context

- 팀이 합의한 rules, templates, commands를 에이전트 입력으로 고정하는 방식.
- 개인 프롬프트보다 버전 관리 가능한 문서와 명령어가 재현성에 유리하다.

## Human Checkpoint

- 각 단계의 출력물을 사람이 승인해야 다음 단계로 넘어가는 구조.
- AI의 비결정성과 맥락 유실을 통제하는 장치.

## Workflow as Interface

- 사람이 자연어로만 지시하는 것이 아니라 파일, 명령어, 템플릿을 통해 에이전트와 상호작용하는 방식.
