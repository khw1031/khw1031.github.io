# Interview Questions

## Concept

- Agentic Coding과 Vibe Coding을 어떻게 구분하는가?
- Human in the Loop은 어떤 단계에서 필요하고, 과도하면 어떤 비용이 생기는가?
- Prompt Drift와 Cascading은 실제 코드베이스에서 어떻게 나타나는가?

## Workflow Design

- 왜 `Research → Plan → Review → Implement → Verify`로 단계를 나누었는가?
- 00-plan.md, 10-task.md, 20-result.md는 각각 어떤 계약을 갖는가?
- 컨텍스트를 새로 여는 전략이 왜 멀티 턴 성능 저하를 줄이는가?

## Team Adoption

- 개인 생산성 도구를 팀 표준으로 바꿀 때 가장 큰 저항은 무엇인가?
- PR에서 워크플로우 준수를 강제하면 어떤 부작용이 생기는가?
- 표준화와 개발자 자율성의 균형은 어떻게 잡는가?
