# CV Project Workspace

이 디렉터리는 CV의 `주요 프로젝트` 후보를 프로젝트별 Markdown으로 정리하는 작업공간이다.

## Workflow

1. 신규 프로젝트는 `_new-project-template.md`를 복사해 시작일 기준 `YYYY-MM-slug.md` 파일로 추가한다.
2. 프로젝트는 오래된 항목부터 하나씩 검토한다.
3. 각 프로젝트에서 부족한 근거, 더 깊게 학습해야 할 영역, 공개 가능한 레퍼런스를 확인한다.
4. 사용자와 질의응답으로 내용을 보강한 뒤 `status`를 `ready`로 바꾼다.
5. `ready` 프로젝트만 `src/data/cv.ts`의 `주요 프로젝트` 섹션에 반영한다.

## Interaction Rule

- 한 번에 하나의 프로젝트만 정리한다.
- 질문은 3개 이하로 묶고, 사용자가 답한 내용만 근거로 반영한다.
- 내부 링크나 비공개 자료는 작성 근거로만 사용하고 공개 CV에는 노출하지 않는다.
- 공개 가능한 링크와 이미지만 `References`에 남긴다.
