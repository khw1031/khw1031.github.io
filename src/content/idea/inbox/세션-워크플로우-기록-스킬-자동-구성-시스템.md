## 아이디어

프로젝트 레벨에서 코딩 에이전트 세션의 진행 과정을 관찰하고, 반복적으로 수행되는 작업(예: Python API + LLM 프로젝트 스캐폴딩, 배포 설정 등)을 자동으로 감지하여 문서화·스킬化する하는构想이다. 단순 매크로 기록을 넘어, 에이전트가 세션 흐름 속에서 "이 패턴은 재현하면 유용하겠다"는 작업을 스스로 식별하고, 훅(hook)과 스킬 정의 형태로 패키징하여 다음 세션에서 즉시 재사용할 수 있게 하는 것이 핵심이다.

기존 프로젝트 템플릿(cookiecutter 등)이 정적 스냅샷이라면, 이 아이디어는 라이브 세션에서 동적으로 패턴을 추출하고 진화시키는 점이 차별점이다. 배포 설정, 의존성 구성, 테스트 구조 잡기 등 프로젝트마다 반복되지만 매번 약간씩 다른 작업들을 스킬로 격상시켜 누적할 수 있다면, 에이전트의 프로젝트 적응 속도가 세션을 거듭할수록 빨라지는 효과를 기대할 수 있다.

## 더 해볼 질문

- "반복되는 작업"을 어떤 기준으로 감지할 것인가? (명시적 사용자 트리거 vs 에이전트 자동 패턴 인식)
- 기록된 스킬은 어떤 포맷으로 저장되며, 프로젝트 간 이식성은 어떻게 확보하는가?
- 훅(hook)은 세션의 어떤 시점(사전/사후/중간)에 개입하며, 기존 pi 스킬 시스템과 어떻게 통합되는가?
- 민감 정보(API 키, 내부 경로 등)가 스킬에 하드코딩되지 않도록 어떻게 필터링할 것인가?
- 스캐폴딩 스킬 생성 시 기존 템플릿 도구(cookiecutter, copier 등)와의 역할 경계는?
- 생성된 스킬의 품질을 검증하고 버전 관리하는 워크플로우는?

## 레퍼런스·서비스

_확인일 2026-07-23. 후보 URL을 실제 fetch해 검증한 빠른 스캔 — 깊은 1차 검증은 하지 않음._

- **[Cookiecutter](https://cookiecutter.readthedocs.io/)** — (1차) · cookiecutter(프로젝트 템플릿)에서 프로젝트를 생성하는 커맨드라인 스캐폴딩 도구.
- **[Copier](https://copier.readthedocs.io/)** — (1차) · (내용 미확인) 프로젝트 템플릿 생성·업데이트 도구이나 fetch 본문에 실질 설명 없음.
- **[GitHub Actions Reusable Workflows](https://docs.github.com/en/actions/sharing-automations/reusing-workflows)** — (1차) · YAML 기반 재사용 가능 워크플로우로 CI/CD 중복을 피하는 공식 패턴 안내.
- **[agent-retro](https://github.com/giannimassi/agent-retro)** — (1차) · AI 코딩 에이전트를 위한 세션 회고(retrospective) 스킬 GitHub 저장소.
- **[NVIDIA NeMo Agent Toolkit — AI Coding Agent Skills](https://docs.nvidia.com/nemo/agent-toolkit/latest/resources/contributing/agent-skills.html)** — (1차) · (내용 미확인) AI 코딩 에이전트 스킬 기여 가이드 페이지이나 fetch 본문에 실질 설명 없음.
- **[MindStudio Session-to-Skill Extractor](https://www.mindstudio.ai/blog/session-to-skill-extractor-agent-procedures)** — (1차) · 에이전트 세션 대화를 추출해 재사용 가능한 스킬(절차)로 전환하는 도구 소개.
- **[prism-mem](https://dev.to/rahul_talatala/prism-mem-automatic-knowledge-extraction-for-ai-coding-agents-2bgo)** — (1차) · 세션 간 상태 없는 코딩 에이전트에 트랜스크립트 411KB를 11.7K 문자로 압축·구조화해 자동 컨텍스트를 제공하는 지식 추출 도구.
- **[Project Templates Claude Code Skill](https://mcpmarket.com/tools/skills/project-scaffolding-templates)** — (1차) · SaaS API, ML 서비스, CLI 도구 등 큐레이션된 템플릿으로 프로젝트 초기 스캐폴딩을 자동화하는 Claude Code 스킬.
